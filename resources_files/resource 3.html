Fiverr project brief for Sidetick website

Project Brief

Sidetick — Website Build (3 pages)

Trading education brand. Indian market. Built from scratch.

About the brand

Sidetick is a systematic trading education brand for Indian retail traders. We teach TradingView-to-broker automation (Pine Script, Dhan API). Our flagship product is Mini Quant — a 22-hour, 11-module course with automation scripts. I have ~40K Instagram followers and a warm WhatsApp/Telegram audience of ~5K. The website needs to convert warm leads and house course content post-purchase.

Pages needed

Page 1 Course landing page
  • Sales page for Mini Quant
  • Razorpay payment button + checkout integration
  • On successful payment: automatically grant access to the member portal (via Razorpay webhook)
Page 2 Gated member portal
  • Only accessible to paying students — no public access
  • Frictionless login
  • Course videos hosted directly on our server or third party tool
  • Videos must be streamed (not downloadable) — no right-click save, no direct URL exposure,
  • Module/lesson navigation with progress tracking
  • Piracy protection is a hard requirement
Page 3 Freebies page
  • Publicly accessible — Need to Enter WhatsApp number to access for FREE
  • Lists free resources (Pine Script templates, strategy guides, etc.)
  • Each freebie has a short description + download/access button
  • Doubles as a lead magnet / trust-building page

Design & tech requirements

Design theme

Light, clean, trustworthy

Payment gateway

Razorpay (India)

Video hosting

Self-hosted on our server or third party tool

Platform preference

Open to AI - Recommend what's fastest

What I need from you (before we start)

  1. Your platform recommendation (WordPress, Webflow, custom, etc.) and reasoning
  2. How you'll handle video streaming + piracy protection — this is the most critical technical piece
  3. How Razorpay payment success will automatically provision access to the member portal
  4. Portfolio samples of membership/gated content sites you've built before
  5. Realistic timeline estimate

What I'll provide

  • Brand assets — logo, colors, fonts
  • All written copy for each page
  • Course videos (uploaded to server once portal is set up)
  • Razorpay API credentials
